Installing Django

To install the latest version of Django, you can use pip (Python's package installer). You will need to have Python and Django is installed using pip, with this command:

(myworld) C:\Users\Your Name>py -m pip install Django

Check Django Version

You can check if Django is installed by asking for its version number like this:

(myworld) C:\Users\Your Name>django-admin --version

If Django is installed, you will get a result with the version number:

4.1.2
